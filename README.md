# Proyecto Web – Formulario, Presentación y Tabla

Este repositorio contiene un proyecto básico en HTML y CSS que incluye varios elementos de práctica: un formulario interactivo, una tabla estructurada, una presentación y un archivo personalizado con mi nombre.

## Estructura del proyecto y escripción de los archivos

- **formulario.html:** Contiene un formulario con distintos tipos de campos (texto, correo, select, etc.) y un botón de envío.  
- **presentacion.html:** Página de presentación personal o de introducción al proyecto.  
- **tabla.html:** Muestra una tabla con datos organizados de forma clara y semántica.  
- **Noemibriz.html:** Archivo personalizado con información o contenido propio.
